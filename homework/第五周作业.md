## 本周作业
打造属于你自己的静态文件服务器,支持以下功能
- 读取静态文件
- MIME类型支持
- 缓存支持/控制
- 支持gzip压缩
- 只能访问指定目录, 不能访问指定目录的上级目录，保证安全 
- 访问目录可以自动寻找下面的index.html文件
- Range支持，断点续传
- 发布为可执行命令并可以后台运行,可以通过`npm install -g`安装
