## 2.用户

### 2.1 登录

#### 2.1.1 请求路径

/login

#### 2.1.2 请求方法

POST

#### 2.1.3 请求参数

放在请求体里

```
{
   "email":"[email protected]",
}

```

### 2.2 验证用户身份

#### 2.2.1 请求路径

/validate

#### 2.2.2 请求方法

POST

#### 2.2.3 请求参数

放在请求体里

```
{
   "token":"xxxxxx",
}

```
